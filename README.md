# poc-BIAN

## Run in development for the first time

### 1. Create environment

Windows:

``` bash
mkdir myproject
cd myproject
py -3 -m venv .venv
```

macOS/Linux:

``` bash
mkdir myproject
cd myproject
python3 -m venv .venv
```

### 2. Activate environment

Windows:

``` bash
.venv\Scripts\activate
```

macOS/Linux:

``` bash
. .venv/bin/activate
```

### 3. Install requirements

``` bash
pip install -r requirements.txt
```
